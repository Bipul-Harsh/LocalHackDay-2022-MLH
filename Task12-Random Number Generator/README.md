# Random Number Generator

Generates a random number between min and max value.

</div>

## To Run the Code

### With Default Values

```bash
python3 main.py
```

### Provinding min and max constraints

```bash
python3 main.py --min <value> --max <value>
```

## Getting Help

```bash
python3 main.py --help
```

## Output

```bash
$ python3 main.py
73
```

```bash
$ python3 main.py --min 23 --max 45
42
```

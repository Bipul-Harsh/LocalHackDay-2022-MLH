## To Generate a Cipher

### With Default Shift

```bash
python3 main.py --text "<text>"
```

### With Custom Shift

```bash
python3 main.py --text "<text>" --shift <shift>
```

## Getting Help

```bash
python3 main.py --help
```

## Output

```bash
python3 main.py --text hello world --shift 1
Cipher: ifmmp xpsme
Shift: 1
```

```bash
python3 main.py --text hello world --shift 2
Cipher: jgnnq yqtnf
Shift: 2
```
